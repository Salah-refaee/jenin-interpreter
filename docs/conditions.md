## switch-case
- Syntax
```
switch (expr) {
  case <expr>: codeblock
  case <expr>: codeblock
  ...
  default: codeblock
}
```
- Example
```
let name = input();
switch (name) {
  case "jenin": {
    println(value:"ME!");
  }
  case "salah": {
    println(value:"MY CREATOR!");
  }
  default: {
    println(value:name);
  }
}
```
